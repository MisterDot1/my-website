# Мир Настольных Игр — демо-сайт

Простой одностраничный сайт для любителей настольных игр. Сделан как лёгкий стартовый шаблон: можно открыть прямо из папки или доработать под свои нужды.

## Как запустить

- **Вариант 1: просто открыть файл**
  1. Открой папку проекта `nasibylin`.
  2. Дважды кликни по файлу `index.html`.
  3. Сайт откроется в браузере (Chrome, Edge и т.п.).

- **Вариант 2: через локальный сервер (рекомендуется для разработки)**
  1. Установи [Node.js](https://nodejs.org).
  2. В терминале/PowerShell перейди в папку проекта:
     ```bash
     cd c:\Users\admin\Desktop\nasibylin
     ```
  3. Установи зависимости:
     ```bash
     npm install
     ```
  4. Запусти простой сервер (используется `serve`):
     ```bash
     npm start
     ```
  5. Открой адрес, который покажет терминал (обычно `http://localhost:3000` или `http://localhost:5000`).

## Структура

- `index.html` — основная страница:
  - герой-секция,
  - блок с подборкой игр и фильтрами,
  - информация про клубы и встречи,
  - гайды для новичков,
  - отзывы игроков.
- `styles.css` — стили, современный тёмный/светлый дизайн, адаптив под мобильные.
- `script.js` — данные по играм, фильтрация, переключение темы, текущий год в подвале.
- `package.json` — настройки npm (локальный сервер).

## Как менять контент

- **Игры** — редактируй массив `games` в `script.js` (названия, описание, тип, сложность и т.д.).
- **Текстовые блоки** — изменяй прямо в `index.html` (заголовки, описания, отзывы).
- **Цвета и размеры** — меняй CSS-переменные и стили в `styles.css`.

Если хочешь, могу помочь адаптировать этот сайт под твой клуб, магазин или личное портфолио настольщика.
